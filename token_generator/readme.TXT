In the Hope.ini uncomment the environment instance that you want to connect(remove "!")

--------------------------------------------------------------------------------
For generating tokens
ensure Service user has create and write access to PCC MajorResource
use the command:

token PROVIDER_NAME PRACTITIONER_IDENTIFIER

PROVIDER_NAME = code of the provider
PRACTITIONER_IDENTIFIER = unique identifier of the practitioner
--------------------------------------------------------------------------------
For displaying the list of all provider for that instance
use the command:

pro
